# Docs

This directory holds repository-level documentation for `greentic-perf`.

Current documents:

- `contribution-contract.md`: scope, governance, fixture/scenario contribution rules, budget guidance, and the difference between PR smoke and nightly runs.
- `fixtures.md`: generated fixture model, source-of-truth answers, and current generator policy.
- `runtime-perf.md`: `gtc start` runtime harness scope, startup validation, and the current Direct Line gating notes.

Keep implementation details in code and tests. Use `docs/` for repo-wide rules, operating guidance, and contributor contracts.
